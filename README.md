# Projeto:

Login usando o JWT e CRUD para cadastro de despesas. 

## Recursos
- Vue 3
- Laravel
- Wsl 2

## Pré-requisitos

- node v14.15.1
- npm 6.14.8
- Laravel Framework 10.13.5
- Vue 3

## Instalação Front

1. cd front
2. npm install.
3. quasar dev.

## Instalação Back

1. cd laravel_onfly
2. npm install.
3. sudo apt update
4. sudo apt install php-curl ( caso ñ tenha a extensao habilitada ) 
5. sudo apt install php-xml  ( caso ñ tenha a extensao habilitada ) 
6. composer install
7. composer update
8. php --ini
9. php artisan serve
